## How to Install?
```
curl https://raw.githubusercontent.com/h0ru/sith/main/files/install.sh | bash
```

### Running
```                 
┌──(kali㉿kali)-[~]
└─$ sith                   #Run the program

[SITH]> ? or help          #Ask for help
```
